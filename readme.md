# FOODOG

Project Foodog using Wordpress to built it.

## Getting Started


### Prerequisites

We need to install Wordpress, WAMP/MAMP/XAMP for the database and VSCODE or PHPStorm

* [Visual Studio Code](https://code.visualstudio.com) - Code Editor
